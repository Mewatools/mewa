◆ parent()

MxWidget * MxWidget::parent ( ) const

Returns the parent of this QxWidget. The parent of a QxWidget is the layout where the widget was added. Top level widgets return QxPainter as parent. QxPainter is the root of all widgets.

The parent is set when a widget is added to a layout.
