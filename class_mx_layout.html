Detailed Description

The MxLayout class is the base class for layout widgets.

MxLayout is the base class for layout widgets. It holds a list of widgets to wich it sets its size and position. It also forwards the draw calls to its child widgets.

To create a custom layout derive the new layout class from MxLayout and overwrite the sizeChanged() function to set the child widgets positions and sizes.
