Detailed Description

Like an SVG drawing, a Mewa SVG is a collection of Mewa triangles that are renderered together as a single draw.

It makes an efficient use of icon drawing, as it doesn't use any texture memory and can be rendered at any size without any performance penalty.

A Mewa SVG is derived from MxAbstractSvg.

See also
DownArrowSvg, HeartSvg and PlusMinusSvg
