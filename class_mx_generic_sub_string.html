Detailed Description

template<typename T>
class MxGenericSubString< T >

This class was introduced to improve performance by avoiding string copies while handling substrings obtained from existing strings. MxGenericSubString holds a const pointer to guarantee a read-only access to string data.

See also
MxSubString, MxSubByteArray and MxGenericString
