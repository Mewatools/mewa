Detailed Description

The MxRenderer keeps track of the rendering state and resources.

The purpose of MxRenderer is to separate the rendering code from the application code, in order to isolate all graphics API calls. This way we can easily support multiple graphics APIs, such as Direct12, Metal, Vulkan and OpenGL.
